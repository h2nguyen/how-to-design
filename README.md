# how-to-design

A growing collection of **free, interactive, multilingual (Deutsch / English / Tiếng Việt), fully-offline courses** that take complete beginners — no design background needed — through **UX/UI and enterprise design**, and into confidently **briefing and reviewing AI** design/front-end collaborators.

Open `index.html`, pick a course, and learn by doing.

## Try it

**Open `index.html`** in any "modern" browser (double-click) — that's the portal. Pick a course; each is fully self-contained (CSS, JavaScript, all languages, and all graphics are inlined), so **no install, no server, and no internet are required**. Switch language top-left; your choice is remembered.

## What makes it different

- **Learn by doing** — clickable drills, a live design-token playground, writing pads with reveal-able model answers, and scored multiple-choice quizzes.
- **A picture for every idea** — a right-hand panel shows a simple illustration that changes as you scroll.
- **Story-driven** — a small recurring cast (Lena, you, Mateo, etc.) makes concepts stick, professionally.
- **Multilingual by design** — German, English, and Vietnamese from one source.
- **One file per chapter** — open a single chapter standalone to focus.
